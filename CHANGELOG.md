# 539 Analyzer 更新記錄

## 2026-03-19 — 侘寂風 UI 大改版

### 設計風格
- 整體轉向**侘寂風（Wabi-sabi）**設計語言
- 色彩收斂為枯金色 `#b8a07e` + 墨灰系，移除霓虹/紫光
- 號碼球從光澤立體改為陶土質感（朱砂紅、墨灰、青石色）
- 背景移除 `backdrop-filter:blur` 霧面效果，改用深色半透明
- 新增 hover 效果：滑鼠移過卡片時微微浮起 + 光暈
- 字體改用 `Noto Serif TC`（襯線體）

### Emoji 精簡
- 全站 Emoji 從 21 個大幅縮減
- 🎯→◆、👑→◈、🗑️→✕、🧰→移除、🔥→·
- 工具箱 Tab：🌟❄️🤖🔢📋🎯 → 精選/冷熱/AI/數據/圖表/追蹤
- 「號碼垃圾桶」→「號碼排除區」
- 「我的專屬工具箱」→「分析工具」

### 冷熱圖簡化
- 移除：推薦/備選/排除 標記、「久」標記、邊框差異
- 只保留三段式顏色（熱號暖色 / 普通 / 冷號灰藍）+ 出現次數
- 圖例簡化為：■ 熱號（前25%）、■ 冷號（後25%）
- 新增說明文字：「近 N 期，顏色越暖 = 出現越多」

### 「為什麼推薦」卡片
- 排版從 `flex` 改為 `CSS Grid`（`auto-fill, minmax(110px, 1fr)`）
- 手機版強制 2 欄，最後一個奇數卡片自動跨欄填滿整行
- 移除 `max-width:160px` 限制

### 手機版 RWD 優化
- 新增 `@media (max-width: 768px)` 完整手機適配
- 推薦卡片取消等高（`min-height:auto`），避免垂直排列時大片空白
- 等高 JS 加入寬度判斷：`clientWidth < 768` 時跳過
- 號碼球、間距、Tab 按鈕、padding 全面縮小適配
- 新增 `@media (max-width: 480px)` 小螢幕進一步優化

### 其他
- 勝率顯示改為全正面四級制（移除「需要調整」負面文字）
- 號碼輸入支援更多分隔符（逗號、頓號、斜線、空格等）
- 對獎動畫延長為 24 幀
- 「TOOL KIT v5」→「v5」
- 「AI 首選」→「首選」
